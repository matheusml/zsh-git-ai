## Quick Start

### ğŸº Install with Homebrew (Recommended)

```bash
# 1. Install via Homebrew
brew tap matheusml/zsh-git-ai
brew install zsh-git-ai

# 2. Add to your shell
echo "source $(brew --prefix)/share/zsh-git-ai/zsh-git-ai.zsh" >> ~/.zshrc
```
